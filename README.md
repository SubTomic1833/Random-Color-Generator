# Random Color Generator

A website where you click to get a random color, set favorites, get achievements, and draw and share your paintings!

Try it for yourself : https://subtomic1833.github.io/Random-Color-Generator/
---

## Features

### Random Color Generator
- Click anywhere on the page to generate a new random color
- Displays color in:
  - HEX
  - RGB
  - HSL

### Copy Color Values
- Click any color value to copy it instantly
- Built-in popup confirmation

### Favorites System
- Save your favorite colors
- Stored using `localStorage` (stays even after refreshing)
- Hover to delete unwanted favorites

### History
- Tracks last 20 generated colors
- Click any color chip to reuse it

### Stats Tracker
- Total colors generated
- Brightest color detected
- Darkest color detected
- Most saturated color detected

---

## Paint Canvas 

- Draw using your current selected color
- Smooth brush strokes
- Toggle **Eraser Mode**
- Save artwork as a PNG image and share with others

### Tools:
- Brush (uses current color)
- Eraser
- Export canvas as image
